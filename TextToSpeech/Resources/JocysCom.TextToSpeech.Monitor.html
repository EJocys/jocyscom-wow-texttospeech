Latest version of WoW “Addon” can read quests and chat messages.
Please, use it with latest version of “Monitor”.

How to check local version of “Jocys.com Text to Speech WoW Addon”...

1. Run “World of Warcraft” game.
2. Open NPC dialogue / quest window.
3. Click on [ TTS... ] button at the bottom of NPC dialogue / quest window.
4. Text-to-speech options window will open.
5. Version number is in the title of text-to-speech options window.
