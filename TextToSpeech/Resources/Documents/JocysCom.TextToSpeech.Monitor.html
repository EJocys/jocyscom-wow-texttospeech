How to check local version of “Jocys.com Text to Speech WoW Addon”...

1. Run “World of Warcraft” game.
2. Move mouse pointer over [■] (Stop) button -- it is near WoW's “Social” button.
3. Click on [≡] (Options) button.
4. Text-to-speech options window will open.
5. Version number is in the title of text-to-speech options window.
